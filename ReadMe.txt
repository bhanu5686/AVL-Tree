prerequisite:
     following software should be installed in system if using terminal:
	1. graphviz
	2. make 
	3. Dev c++ or Codeblocks ide
	

	steps to run the program in ide:
	   1. open the main.cpp file in ide
	   2. run and compile


	main.cpp can also be executed using terminal.
	steps to run the program in terminal:
	   1. open terminal and drive into this directive
	   2. type the commands
		make  // this will compile the .cpp file
		make run   // this will execute the compiled file
